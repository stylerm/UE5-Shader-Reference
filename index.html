What's your bottleneck?
🔥
Translucency / overdraw
Keep translucent under 3ms. Cutouts, unlit, smaller sprites.
 🧨
Shader recompile cascade
MPC changes invalidate every material that samples them. Fix the graph, not the param.
 🌫
Scalability pop-in / LOD lag
Effect Type + Significance Handler must both be set. Update freq = Low or Medium.
 🧵
Dispatch > 65,535 threads
GPU sim silently remaps. Plan thread count around the D3D limit.
 🧬
Custom Node dedup fails
Identical HLSL with different input wiring compiles twice. Normalize inputs.
