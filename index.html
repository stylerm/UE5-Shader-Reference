GLSL → HLSL Intrinsic Mapping · Always Ask Claude to Use HLSL Intrinsics

Custom Node Rules
Custom Node · Never Do This
✗No #include — body is injected mid-shader, includes break compilation
✗No function declarations with signatures — inline all logic
✗No sampler declarations — pass textures as input pins instead
✗No global-scope mutable state — all vars are local to the body
✗No GLSL precision qualifiers (mediump, highp, lowp)
✗No void main() — body only, return value is the output
✗No undeclared variable references — only use explicitly listed input pins
Custom Node · Ask Claude For These
✓Inline-only logic — no separate function blocks
✓static const for all constants
✓Explicit return value matching declared return type
✓Swizzle count validation — float3 output ≠ float4 return
✓Step-by-step comments on each major operation
✓Feature level guard hints if targeting ES3.1 / mobile
✓Flag texture samples as [NEEDS TEXTURE PIN] rather than inline
.usf / .ush File Rules
✓Ask for include guards (#pragma once)
✓Ask for /Engine/Public/Platform.ush base include
✓Declare module macro: YOURMODULE_API
✓Use virtual shader path mapping for plugin custom paths
!SM6 only: declare WaveOps / RayQuery usage explicitly
!Request #if FEATURE_LEVEL >= ... guards for cross-platform

Task: Port the following GLSL to a UE5 Custom node Source: [paste GLSL shader here] Apply ALL conversions: mix → lerp fract → frac mod → fmod atan → atan2 dFdx/y → ddx/ddy vec2/3/4 → float2/3/4 mat3/4 → float3x3/float4x4 iTime → Time (float input pin) iResolution → ScreenSize (float2 input pin) gl_FragCoord → use a ScreenPos input pin Output rules: - Body-only, no function signature - No sampler declarations - Flag texture samples as [NEEDS TEXTURE PIN] - Inline all helper functions - Add a comment for each conversion made

Common Errors to Prevent
!Matrix multiply order — HLSL is row-vector × matrix; GLSL is column-major
!UV origin — UE flips V; ask Claude to account for 1.0 - UV.y if porting
!Swizzle mismatch — always ask Claude to verify component counts explicitly
!Shadertoy time is iTime — must be mapped to a float Time input pin
!Integer division — 1/2 = 0; cast to float explicitly: 1.0/2.0
!No implicit broadcast — float3(1) fails; use float3(1,1,1)
